# Identifying Sentiments of a Paragraph (Semantic Analysis)

An interactive Sentiment Analysis Model that portrays the sentiment of text and provides comparative analysis of ressults among the five proposed models (SNN, CNN, LSTM, BILSTM and an auto assembled Hybrid Neural Network).
